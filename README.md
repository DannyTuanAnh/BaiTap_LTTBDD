## Công nghệ sử dụng

- **Language:** Dart
- **Framework:** Flutter
- **IDE:** Visual Studio Code / Android Studio
- **Version Control:** Git

## Cách sử dụng

1. **Clone repository:**

   ```bash
   git clone https://github.com/DannyTuanAnh/BaiTap_LTTBDD.git
   ```

2. **Di chuyển vào thư mục bài tập cần chạy:**

   ```bash
   cd <tên_thư_mục_bài_tập>
   ```

3. **Cài đặt dependencies:**

   ```bash
   flutter pub get
   ```

4. **Chạy ứng dụng:**
   ```bash
   flutter run
   ```

## Ghi chú

- Mỗi thư mục chứa một project Flutter độc lập
- Đảm bảo đã cài đặt Flutter SDK trước khi chạy
- Kiểm tra file `pubspec.yaml` trong mỗi thư mục để biết các dependencies cần thiết
